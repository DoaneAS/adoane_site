# Images Directory

## Required Image

### Professional Headshot
**File**: `headshot.jpg`

**Specifications**:
- Format: JPG or PNG
- Dimensions: 400x400px to 800x800px (square, 1:1 aspect ratio)
- File size: < 200KB (optimized for web)
- Quality: Professional photo with clear face visibility
- Background: Preferably neutral or professional setting

**How to Add**:
1. Choose your professional photo
2. Resize to square dimensions (e.g., 600x600px)
3. Optimize for web using tools like:
   - TinyPNG (https://tinypng.com)
   - ImageOptim (macOS)
   - Squoosh (https://squoosh.app)
4. Save as `headshot.jpg` in this directory
5. Commit and push to GitHub
